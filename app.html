Web app for visualising uncertainty in climate-sensitive epidemiological projections (hosted on Hugging Face Spaces). Note that the web app is only intended for exploratory purposes and is not able to handle multiple users simultaneously. For more advanced usage, consider running the app locally or using the accompanying Python package (see the Python package documentation).
